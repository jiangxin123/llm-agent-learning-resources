### 中文长版

`llm-agent-learning-resources` 正式发布了。

这是一个面向中文开发者的大模型、RAG 与 Agent 学习资源导航仓库，目标不是单纯堆链接，而是尽量把“值得看什么、适合谁看、应该怎么学”整理清楚。

- LLM 基础
- RAG / 知识库

如果你也在学大模型应用、知识库、Agent 或工作流系统，欢迎来看看，也欢迎推荐值得收录的资源。
